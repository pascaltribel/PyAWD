Marmousi field¶

This is an example of wave propagation speed profile which can be used as is in pyawd.

Libraries import¶

In [8]:
from pyawd import Marmousi

The profile can be instanciated using the Marmousi constructor, which allows specifying a resolution.

In [9]:
marmousi = Marmousi(1024)
marmousi.plot()
